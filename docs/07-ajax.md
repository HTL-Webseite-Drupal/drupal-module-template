# AJAX

AJAX is used to call the REST endpoint from the frontend.

Flow:
Twig → JavaScript → REST API → Service → JSON → UI

This example shows:
- Drupal behaviors
- Library attachment
- Simple jQuery usage

## Example

The example demonstrates how to:
- Attach a library to a Twig template
- Use a Drupal behavior to initialize JavaScript
- Make an AJAX request using jQuery
- Handle the response and update the UI
